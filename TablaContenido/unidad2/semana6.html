Unidad 2

Semana 14

Temas:

  • Registro de Microservicios en Eureka
  • Creación del servidor Eureka
  • Consumo de un Microservicio
  • Configuración avanzada en Eureka
  • Configuración en la nube

Definiciones:

1. Registro de Microservicios en Eureka: Proceso de inscribir un microservicio en el servidor Eureka para que esté disponible para otros servicios.
2. Creación del servidor Eureka: Configuración inicial de un servidor Eureka que actúa como registro centralizado para los microservicios.
3. Consumo de un Microservicio: Acción de utilizar y acceder a un microservicio registrado en Eureka desde otra aplicación.
5. Configuración avanzada en Eureka: Ajustes detallados para optimizar y personalizar el funcionamiento del servidor Eureka.
5. Configuración en la nube: Adaptación de la infraestructura Eureka para despliegue y gestión eficientes en entornos de nube.

Procedimiento:

° Implementé eventos, condicionales y bucles en aplicaciones web para mejorar la interacción del usuario.
° Utilicé renderizado condicional e iterativo para mostrar contenido de manera dinámica en respuesta a eventos o condiciones.
° Diseñé formularios y gestioné el enrutamiento para ofrecer una experiencia de usuario más fluida.
° También realicé el consumo de APIs, aplicando conceptos de Promesas y Async/await para obtener datos de forma asincrónica, y empleé la librería Axios para simplificar la comunicación con servidores externos.

Resultado:

° Registro de Microservicios en Eureka: Envia una solicitud al servidor Eureka proporcionando detalles del microservicio para su registro.
° Creación del servidor Eureka: Instala Eureka, configura las propiedades básicas y inicia el servidor para habilitar el registro de microservicios.
° Consumo de un Microservicio: Consulta el registro Eureka, identifica el microservicio necesario y configura la aplicación para consumirlo.
° Configuración avanzada en Eureka: Ajusta las propiedades de configuración en el archivo de configuración de Eureka según los requisitos específicos.
° Configuración en la nube: Implementa el servidor Eureka en una plataforma en la nube, ajustando la configuración para aprovechar características específicas del entorno.
