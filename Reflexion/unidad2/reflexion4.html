Semana 12

Reflexión

Durante la realización del laboratorio 12, fortalecí mi comprensión del desarrollo backend y la arquitectura de aplicaciones web.

La exploración de la funcionalidad del lado del servidor, el uso de servidores web como Tomcat, y la manipulación de Java y Jakarta en IntelliJ Idea me brindaron la oportunidad de profundizar en el desarrollo de aplicaciones robustas.

¿Que aprendí?

Adquirí conocimientos acerca de los conceptos fundamentales de Spring Core y Spring Boot, comprendiendo la terminología asociada y las características distintivas de estos marcos.

La implementación de Servlets y JSP proporcionó una visión práctica, y el enfoque en Spring Boot me permitió desarrollar aplicaciones backend eficientes y escalables.

¿Cómo aprendí?

La combinación de teoría y aplicación práctica, haciendo uso de herramientas como IntelliJ Idea y Tomcat, resultó fundamental para asimilar los conceptos.

Además, la ejecución de un caso práctico de consumo de API en el desarrollo de un sitio web backend con Spring Boot consolidó mi capacidad para aplicar estos conocimientos en situaciones del mundo real.
