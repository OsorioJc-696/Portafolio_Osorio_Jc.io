Reflexión

En el proceso de desarrollo de la API Rest con Spring, consolidé mis conocimientos en la arquitectura de APIs Rest y el patrón MVC. Este laboratorio me permitió aplicar conceptos de HTTP y entender cómo diseñar servicios web de manera eficiente.

Abordar desafíos prácticos durante la implementación proporcionó una comprensión más profunda de cómo funcionan las APIs Rest en un entorno de desarrollo real.

¿Que aprendí?

Aprendí a implementar servicios RESTful utilizando Spring y a diseñar una arquitectura escalonada para una aplicación web. Gané experiencia en la manipulación de solicitudes HTTP, la estructuración de rutas y la creación de controladores.

La integración con Spring MVC fortaleció mi comprensión de cómo se gestionan las interacciones entre el cliente y el servidor.

¿Cómo aprendí?

El enfoque de aprendizaje se centró principalmente en la práctica, participando activamente en el desarrollo de la API y resolviendo desafíos en tiempo real. Consulté la documentación de Spring Framework, participé en debates en línea y adopté enfoques iterativos para mejorar la implementación.

La combinación de práctica directa y el uso de recursos en línea resultó fundamental para obtener una comprensión integral del desarrollo de API Rest con Spring.
