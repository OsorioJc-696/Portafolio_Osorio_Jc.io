Definiciones:

1. Desarrollo Backend: La creación de la parte lógica y funcional de una aplicación web, enfocada en el servidor, que maneja la lógica de negocio y la interacción con la base de datos.
Arquitectura de Aplicaciones Web: La estructura organizativa que define la interacción y distribución de componentes en una aplicación web, incluyendo el backend, frontend y la comunicación entre ellos.
Servidores Web: Software que responde a solicitudes de clientes y proporciona recursos estáticos o dinámicos en la web.
Funcionamiento del Server Side: La ejecución de código en el servidor para procesar solicitudes y generar dinámicamente el contenido que se enviará al cliente.
Java y Jakarta: Lenguaje de programación y conjunto de tecnologías para el desarrollo de aplicaciones empresariales en el contexto de la plataforma Java.
Herramientas: IntelliJ Idea, Tomcat - Entornos de desarrollo y servidor web utilizados en el desarrollo backend.
Spring Core y Spring Boot: Framework de desarrollo de aplicaciones Java que facilita la creación de componentes reutilizables y el desarrollo de aplicaciones empresariales.
Terminología y Características de Spring: Conjunto de conceptos y propiedades asociadas con el framework Spring, como inversión de control (IoC), inyección de dependencias y aspectos.
Módulos Spring, Servlets, JSP: Componentes y tecnologías utilizadas en el desarrollo web con Spring, incluyendo la creación de Servlets y páginas JSP.
Spring Boot: Extensión de Spring que simplifica el desarrollo de aplicaciones mediante la configuración automática y el empaquetamiento de aplicaciones en un formato autoejecutable.
Laboratorio 12: Desarrollo de un Sitio Web Backend: Actividad práctica centrada en la creación de un sitio web backend, aplicando los conocimientos adquiridos en el curso.

Procedimiento:

° Crear una página web de Front-End: Diseñar la interfaz de usuario y escribir el código Javascript para interactuar con ella.
° Utilizar intérpretes de Javascript: Ejecutar código Javascript en intérpretes como V8 para visualizar resultados en un navegador.
° Manipular variables y operadores: Declarar variables, asignar valores y efectuar operaciones aritméticas y lógicas.
° Implementar estructuras de control y bucles: Emplear instrucciones condicionales como "if" y bucles como "for" para gestionar el flujo de ejecución.
° Generar funciones: Establecer funciones con la sintaxis adecuada y llamarlas para realizar tareas específicas.
° Explorar funciones avanzadas: Utilizar funciones anónimas, objetos de función, funciones flecha, autoejecutables y cierres para abordar situaciones complejas.
° Trabajar con clases y objetos Number, String y Math: Aplicar funcionalidades avanzadas a números y cadenas, y llevar a cabo operaciones matemáticas complejas.
° Administrar arreglos: Almacenar y manipular datos en arreglos, ya sean unidimensionales o multidimensionales.
° Utilizar objetos Map: Crear y gestionar objetos Map para almacenar pares clave-valor y acceder eficazmente a los datos.

Resultado:

° Desarrollo exitoso de un sitio web backend funcional.
° Aplicación de conceptos de Spring Core y Spring Boot.
° Implementación de Servlets y JSP para la interacción con el usuario.
° Uso eficiente de IntelliJ Idea y Tomcat en el entorno de desarrollo.
° Adquisición de habilidades en el manejo de servidores web y arquitecturas de aplicaciones web.
